# Локальная разработка (Docker)

Здесь находятся файлы для запуска проекта на локальной машине без необходимости поднимать Kubernetes.

## Использование
Главный файл — `docker-compose.yml`. Он поднимает:
* PostgreSQL 16
* Redis 7
* RabbitMQ 3
* Локальные инстансы микросервисов (после того как они будут написаны)

**Команда для запуска локального окружения:**
```bash
docker compose up -d
