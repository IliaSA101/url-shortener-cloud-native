# Микросервисы (Services)

В этой директории располагается исходный код всех микросервисов проекта.

## Компоненты

1. **Gateway (`/gateway`)**
   * **Стек:** PHP / Laravel
   * **Роль:** Публичный API, аутентификация пользователей, веб-интерфейс для генерации ссылок. Отвечает за запись новых ссылок в БД (PostgreSQL) и прогрев кэша (Redis).

2. **Redirector (`/redirector`)**
   * **Стек:** Python / FastAPI
   * **Роль:** Hot path сервиса. Обрабатывает входящие короткие ссылки, ищет оригинал в Redis (fallback в PostgreSQL) и выполняет 302 редирект. Публикует события о кликах в RabbitMQ.

3. **Analytics Worker (`/analytics-worker`)**
   * **Стек:** Python
   * **Роль:** Асинхронный фоновый воркер. Читает очередь событий из RabbitMQ и сохраняет статистику переходов (IP, User-Agent, Referer) в PostgreSQL для дальнейшей агрегации в BI.
