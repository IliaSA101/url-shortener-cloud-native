# Infrastructure as Code (IaC)

Эта директория содержит код для автоматизированного развертывания и конфигурации серверов. Инфраструктура разворачивается с нуля до готового K3s-кластера в два этапа.

## Структура
* **`/terraform`** — провиженинг облачных ресурсов. Поднимает 3 VDS-сервера и настраивает приватную сеть между ними.
* **`/ansible`** — конфигурация ОС на серверах. Устанавливает базовые утилиты, Docker, настраивает безопасность (UFW, SSH) и разворачивает K3s-кластер (1 Master + 2 Workers).
